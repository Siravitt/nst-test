# Project nst-test

## Installation

* Clone the repository:
	* ```git clone https://github.com/Siravitt/nst-test.git```

* Navigate to the project directory:
	* ```cd nst-test```

* Install the project dependencies:
	* ```npm install```

## Testing
	npm test

## Coding Conventions
* Use [jest](https://jestjs.io/) for testing
